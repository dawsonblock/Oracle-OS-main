# Project Memory Index

This directory stores repo-local engineering memory that helps future work stay consistent with prior decisions and known constraints.

## Sections

- [architecture-decisions/README.md](architecture-decisions/README.md) — accepted and draft architecture decisions
- [known-good-patterns/README.md](known-good-patterns/README.md) — reusable patterns backed by repeated success
- [open-problems/README.md](open-problems/README.md) — unresolved technical issues and investigations
- [rejected-approaches/README.md](rejected-approaches/README.md) — paths tried and deliberately avoided
- [risk-register.md](risk-register.md) — active risks and mitigations
- [roadmap-state.md](roadmap-state.md) — current roadmap framing

## Usage Notes

- Keep this material concise and decision-oriented.
- Prefer linking evidence, tests, or traces when a record justifies a decision.
- Use `docs/` for public status/governance narrative and `ProjectMemory/` for engineering memory.
