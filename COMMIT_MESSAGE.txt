feat(runtime): complete single-hard-path kernel rebuild (all 7 phases)

SUMMARY
-------

This commit implements the complete minimal clean rebuild of the Oracle OS
runtime into a true single-hard-path kernel. All 7 phases are complete:

  Phase 1: Execution boundary closed (no .shell, all typed)
  Phase 2: Planner collapsed to single entry (one plan() method)
  Phase 3: State construction centralized (RuntimeBootstrap only)
  Phase 4: File mutations typed (FileMutationSpec through executor)
  Phase 5: Boundary enforcement hardened (runtime assertions + tests)
  Phase 6: Durability strengthened (fsync WAL, determinism tests)
  Phase 7: Artifacts isolated (no alternate paths remain)

ARCHITECTURE
------------

Unified Execution Spine (Guaranteed):

  Intent
    ↓
  RuntimeOrchestrator.submitIntent() ← ONLY INGRESS
    ↓
  Planner.plan() ← SINGLE PLANNER
    ↓ Returns: Command (typed, no .shell)
    ↓
  VerifiedExecutor.execute() ← ONLY EXECUTOR
    ↓ Guards: type✓ policy✓ pre✓ post✓
    ↓
  CommandRouter
    ↓
  TypedRouter (CodeRouter/SystemRouter/UIRouter)
    ↓
  WorkspaceRunner.run<Op>()
    ↓
  DefaultProcessAdapter ← ONLY Process() LOCATION
    ↓
  CommitCoordinator.commit() ← ONLY STATE MUTATION
    ↓ fsync-safe WAL + event log
    ↓
  WorldStateModel (immutable snapshot)
